# EXIFTOOL-Android

Android应用程序，用于查看和修改图像的EXIF数据。

## 功能特性

- 查看图像的详细EXIF信息
- 修改和编辑EXIF数据
- 支持多种图像格式
- 简洁直观的用户界面

## 技术栈

- Kotlin
- Android SDK
- ExifInterface

## 安装方法

1. 克隆仓库：`git clone https://github.com/bluebighead/EXIFTOOL-Android.git`
2. 在Android Studio中打开项目
3. 构建并运行应用程序

## 使用说明

1. 启动应用程序
2. 选择要查看的图像文件
3. 浏览EXIF信息
4. 根据需要编辑EXIF数据
5. 保存更改
